# Ejecutar en desarrollo


1. Clonar el repositorio
2. Ejecutar 
```
yarn install
```
3. Tener Nest CLI instalado
```
npm i -g @nestjs/cli
```

4. Levantar la base de datos
```
docker-compose up -d
```

4. Recargar la base de datos
```
http://lobo.local:3001/api/v2/seed
```

## Stack Usado
* MongoDB
* Nest
